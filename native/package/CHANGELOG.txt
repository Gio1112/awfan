awfan 1.1.1
===========

Fixed
-----
- Replaced the updater's `Get-FileHash` dependency with a direct .NET SHA-256
  implementation.
- Restored updates on Windows installations where the
  Microsoft.PowerShell.Utility module is unavailable or not auto-loaded.
- Aligned the internal awfan-core version with the public frontend version.

Updater compatibility
---------------------
The updater still verifies the downloaded release ZIP against its published
SHA-256 checksum before installing it. The verification now uses
System.Security.Cryptography.SHA256 and works in both Windows PowerShell 5.1
and PowerShell 7 without relying on `Get-FileHash`.

Bootstrap note for awfan 1.0.1
------------------------------
An already-installed 1.0.1 updater contains the old hashing command. Replace
its update.ps1 once with the current release copy, or install 1.1.1 manually.
Future updates will use the fixed updater.

Unchanged
---------
- The secure per-user elevated background broker from 1.1.0 remains included.
- Every hardware-changing command still requires --yes.
- Firmware boost values remain raw inputs rather than percentages or target
  RPM values.
