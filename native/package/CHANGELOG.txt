awfan 1.1.4
===========

Fixed
-----
- Allowed awfan profile 0 --yes to select the manual profile shown by awfan profiles.
- Kept awfan auto restricted to dynamic firmware profiles 1 through 5.
- Updated CLI help, completion, documentation, and Windows parser tests.
- Centralized frontend and core version generation through the CMake project version.

Update
------
Existing installations can install this release with:

   awfan update
